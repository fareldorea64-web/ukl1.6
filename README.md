Program ini digunakan untuk menjumlahkan dua buah matriks dengan ukuran yang sama.
Pertama, pengguna diminta memasukkan jumlah baris dan kolom matriks, lalu mengisi setiap elemen dari matriks pertama dan kedua.
Setelah itu, program melakukan proses penjumlahan dengan menjumlahkan elemen yang berada pada posisi yang sama di kedua matriks.
Hasilnya disimpan ke dalam matriks baru, kemudian ditampilkan ke layar dalam bentuk matriks.

jadi Program ini memudahkan pengguna untuk menghitung hasil penjumlahan dua matriks secara cepat dan otomatis.
